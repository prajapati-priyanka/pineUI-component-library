Image

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Responsive Image

Add the img-responsive class to the image tag. The image will grow to the 100% width of it's parent container.

Round Image

Add the img-round class to the image tag. The image will grow to the 100% width of it's parent container.
