Input

The input tag specifies an input field where the user can enter data. It is the most important form element.The input element can be displayed in several ways, depending on the type attribute.

Text Box

Example of different input types are as follows: type="text" (default value), type="number", type="password", type="date"

Input Validation

Input validation is performed to ensure only properly formed data is entering the workflow in an information system. There are 3 types of inputs available apart from the default one: error, warn, success
